# User Guide Overview

This guide helps you navigate SGM's features for managing sales compensation governance. Whether you're designing policies, processing approvals, resolving disputes, or monitoring compliance, SGM organizes your work into intuitive operational modes.

## How SGM Organizes Work

SGM uses **four operational modes** to organize different types of governance activities:

<CardGroup cols={2}>
  <Card title="Design Mode" icon="compass-drafting" href="/user-guide/modes/design-mode">
    Build governance frameworks, policies, and templates. For consultants, CoE teams, and policy architects.
  </Card>
  <Card title="Operate Mode" icon="gear" href="/user-guide/modes/operate-mode">
    Manage documents, approvals, and daily operations. For compensation teams and RevOps.
  </Card>
  <Card title="Dispute Mode" icon="scale-balanced" href="/user-guide/modes/dispute-mode">
    Handle exceptions, disputes, and case resolution. For sales field staff and case workers.
  </Card>
  <Card title="Oversee Mode" icon="eye" href="/user-guide/modes/oversee-mode">
    Monitor compliance, committees, and governance health. For executives and compliance officers.
  </Card>
</CardGroup>

## Key Features

| Feature | What You Can Do | Mode |
|---------|-----------------|------|
| [Governance Analysis](/user-guide/governance-analysis) | Upload documents and identify compliance gaps | Design |
| [Policy Library](/user-guide/policy-library) | Browse and manage 17+ governance policies | Design |
| [Plan Management](/user-guide/plan-management) | Create and track compensation plans | Operate |
| [Approvals](/user-guide/approvals) | Process approval requests with SLA tracking | Operate |
| [Documents](/user-guide/documents) | Manage governance documents with versioning | Operate |
| [Search & Reports](/user-guide/search-reports) | Find information and generate reports | All |

## User Roles

Your role determines what you can access in SGM:

| Role | Design | Operate | Dispute | Oversee |
|------|--------|---------|---------|---------|
| **SUPER_ADMIN** | Full | Full | Full | Full |
| **ADMIN** | Full | Full | Full | Full |
| **MANAGER** | Read-only | Full | Full | - |
| **USER** | - | Limited | Limited | - |
| **VIEWER** | - | Read-only | - | - |

## Getting Help

### AI Assistants

SGM includes AI-powered assistants to help you:

- **Ask**: Query governance policies in natural language
- **Ops**: Get pattern detection and anomaly alerts
- **Pulse**: Receive daily coaching recommendations
- **Task**: Manage your work queue

Click any orb in the dock to start a conversation.

### Knowledge Base

The Knowledge Base contains 929 reference cards across 8 SPM pillars. Search for any governance topic to find definitions, best practices, and related concepts.
