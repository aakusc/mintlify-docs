# Admin Guide Overview

This guide covers system administration tasks for SGM, including setup, user management, tenant configuration, and integrations.

## Who This Guide Is For

This guide is for:
- **System administrators** deploying and maintaining SGM
- **IT teams** managing integrations and infrastructure
- **Operations leads** configuring workflows and tenants

| Capability | SUPER_ADMIN | ADMIN |
|------------|-------------|-------|
| System setup | ✅ | - |

| Tenant configuration | ✅ | Own tenant only |

## Architecture Overview

SGM uses a **Contracts + Ports + Bindings** architecture:

| Layer | Purpose |
|-------|---------|
| **Contracts** | Zod schemas defining data structures |
| **Ports** | Service interfaces |
| **Bindings** | Provider implementations |

### Binding Modes

| Mode | Data Source | Use Case |
|------|-------------|----------|
| **Synthetic** | In-memory demo data | Development, demos |
| **Mapped** | External API adapters | Integrations |
| **Live** | Prisma database | Production |

Set via `BINDING_MODE` environment variable.

## Technology Stack

| Component | Technology |
|-----------|------------|
| Framework | Next.js 14+ (App Router) |
| Language | TypeScript |
| Database | SQLite (local) via Prisma |
| Authentication | NextAuth |
| Styling | Tailwind CSS |
| Validation | Zod |

## Key Directories

```
app/
├── api/              # API routes
├── (app)/            # Application pages
└── auth/             # Authentication pages

lib/
├── contracts/        # Zod schemas
├── ports/            # Service interfaces
├── bindings/         # Provider implementations
├── config/           # Configuration
└── services/         # Business logic

prisma/
├── schema.prisma     # Database schema
└── data/sgm.db       # SQLite database
```

## Configuration Files

| File | Purpose |
|------|---------|
| `.env` | Environment variables |
| `prisma/schema.prisma` | Database schema |
| `lib/config/` | Application configuration |

## Default Ports

| Service | Port | Notes |

| SGM App | 3030 | Main application |
